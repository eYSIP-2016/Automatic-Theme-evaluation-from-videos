1.Open Matlab
2.Run transforVideo.m from the folder Homography
3.A dialog box will pop up asking to select the desired video.
4.Homographed output will be generated and stored in destination Homography\output.
5.Run VideoEvaluation.m from the folder Video Tracker.
6.A dialog box will pop up asking to select the desired video. Select the homgraphed video from the output folder in destination Homography\output.
7.Run ampli.m from the folder Sound Processing.
8.A dialog box will pop up asking to select the desired video. Select the original video containig the sound.
9.Respective log files will be generated in Text Files Folder.
